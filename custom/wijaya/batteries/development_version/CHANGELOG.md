# Wijaya Internal Development Changelog

This changelog tracks Wijaya Corp's internal development version, which is
versioned independently from official Chatwoot releases. See `README.md` for the
versioning rules and bump workflow.

## v0.1.0 - 2026-06-29

- Introduce internal Wijaya semantic versioning, decoupled from official Chatwoot
  versioning. The internal dev version is displayed on General Settings alongside
  the official Chatwoot version and build SHA.
